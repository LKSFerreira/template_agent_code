# Propósito

Use esta skill quando a resposta depender de informação atual, documentação externa, preços, versões, APIs ou fatos com chance razoável de mudança.

# Regras

- Priorize documentação oficial e fontes primárias.
- Cite links usados na resposta.
- Para dúvidas técnicas, prefira docs oficiais, repositórios oficiais e RFCs.
- Para produtos, leis, preços e versões, confirme a data da fonte.
- Diferencie fato verificado de inferência.

# Restrições

- Não use posts aleatórios como fonte principal para decisão técnica crítica.
- Não copie trechos longos de fontes externas.
