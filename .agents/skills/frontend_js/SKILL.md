# Propósito

Use esta skill para tarefas de UI, React, Vue, Vite, componentes, estilos, acessibilidade e interação.

# Diretrizes

- Respeite o design system e padrões existentes.
- Priorize interfaces densas, claras e previsíveis para ferramentas operacionais.
- Use componentes e ícones já presentes no projeto.
- Garanta responsividade sem sobreposição de texto.
- Trate loading, empty state, erro e sucesso quando fizer parte do fluxo.

# Validação

- Rode build/lint/testes de frontend quando disponíveis.
- Para mudanças visuais relevantes, valide em viewport desktop e mobile quando houver ferramenta de preview.
- Verifique estados interativos, foco e mensagens de erro.

# Restrições

- Não adicione bibliotecas visuais novas sem necessidade real.
- Não crie landing page quando o pedido for por aplicativo, ferramenta ou fluxo funcional.
