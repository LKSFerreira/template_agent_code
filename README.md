# Template Agent Code

Repositório modelo com documentação e templates para desenvolver agentes (MCP, servidores, skills e integrações). Este conjunto de arquivos serve como base e guia para implementação, configuração e integração com ferramentas como VS Code e GitHub Actions.

## Sumário

- **Visão geral:** breve descrição do projeto e objetivos.
- **Estrutura:** lista dos documentos principais do repositório.
- **Como usar:** passos rápidos para começar.
- **Contribuição:** como colaborar.

## Pré-requisitos

- VS Code (recomendado: abrir no Dev Container se disponível).
- Docker (quando usar containers de desenvolvimento).
- Node.js / Python / outras runtimes conforme o agente implementado — ver arquivos específicos para detalhes.

## Como começar

1. Abra o repositório no VS Code:

```bash
code .
```

2. Consulte o índice para localizar o conteúdo desejado: [00_indice.md](00_indice.md).

3. Se houver um Dev Container configurado, use **Reopen in Container** no VS Code.
