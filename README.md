# 🚀 LKSFerreira - Fábrica de Templates

Este repositório é o núcleo de padronização para todos os meus ambientes de desenvolvimento. Ele consolida regras de IA (Antigravity/Gemini), automações (Skills) e fluxos de trabalho (Workflows) unificados.

## 🎯 Propósito
Centralizar a evolução de regras de código e comportamento de agentes IA em um único lugar, permitindo a distribuição rápida para novos projetos ou labs de estudo.

## 🛠️ Como usar (Inicialização)
Para levar estas configurações para um novo diretório, execute o script de inicialização:

```bash
./init.sh
```

### Opções de Contexto:
1.  **Projeto:** Configura o ambiente com foco em qualidade de produção, arquitetura sênior e Clean Code rigoroso.
2.  **Estudo:** Configura o ambiente com foco em aprendizado, explicações detalhadas e documentação didática.

## 🏗️ Estrutura do Repositório
- `.agents/rules/`: Regras de linguagem e comportamentos globais.
- `.agents/rules/projeto/`: Arquivos específicos (AGENTS.md, README.md, workflow.md) para o modo profissional.
- `.agents/rules/estudo/`: Arquivos específicos para o modo de aprendizado.
- `.agents/skills/`: Automações complexas compartilhadas.
- `.agents/templates/`: Modelos de arquivos (Docker, config, etc) usados pelos fluxos de setup.
